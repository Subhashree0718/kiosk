# ── Installation & Setup ─────────────────────────────────────────────────────
# 1. Clone repository
#    git clone https://github.com/Subhashree0718/kiosk.git
#    cd kiosk

# 2. Install all dependencies (all workspaces)
#    npm install

# 3. Set up environment variables
#    cp .env.example .env   (edit DATABASE_URL, JWT_SECRET etc.)

# 4. Run database migrations
#    cd apps/backend
#    npx prisma migrate dev --name init
#    npx prisma db seed     (optional: seed sample data)

# 5. Start all services (Turborepo parallel dev)
#    npm run dev
#    ├── Kiosk Frontend → http://localhost:5173
#    ├── Admin Dashboard → http://localhost:5174
#    └── Backend API    → http://localhost:3001

# 6. Individual app start (optional)
#    npm run dev:kiosk     (kiosk only)
#    npm run dev:admin     (admin only)
#    npm run dev:backend   (backend only)
